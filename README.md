 Customer Retention & Churn Analysis

 📌 Project Overview

This project analyzes customer churn and retention patterns using the Telco Customer Churn dataset. The dashboard helps businesses identify customer behavior, understand churn drivers, and improve customer retention strategies.

 🎯 Objectives

- Analyze customer churn
- Identify retention drivers
- Compare churn across customer segments
- Discover patterns based on contract type and internet service
- Provide business recommendations

 📂 Dataset

Telco Customer Churn Dataset

 🛠 Tools Used

- Microsoft Excel
- Power BI Desktop

📊 Dashboard Includes

- Total Customers
- Churned Customers
- Retained Customers
- Churn Rate %
- Total Revenue
- Average Monthly Charges
- Average Tenure
- Churn by Contract
- Churn by Internet Service
- Churn by Payment Method
- Churn by Gender
- Customer Segmentation
- Interactive Filters

 💡 Key Insights

- Month-to-month contracts have the highest churn.
- Customers using Fiber Optic internet are more likely to churn.
- Electronic Check users show higher churn.
- Long-tenure customers are less likely to leave.
- Annual and two-year contracts improve customer retention.

 📈 Recommendations

- Promote yearly subscription plans.
- Improve support for Fiber Optic users.
- Reward loyal customers.
- Offer retention discounts for month-to-month customers.
- Encourage automatic payment methods.
